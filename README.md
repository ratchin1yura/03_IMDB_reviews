# 03_IMDB_reviews
Analysis for (IMDB reviews)

All data used for educational purposes.

Download archive here (aclImdb_v1.tar.gz): http://ai.stanford.edu/~amaas/data/sentiment/
Unpack archive (folder aclImdb), copy entries in project folder.
Folder contains: test, train, README, imdbEr.txt, imdb.vocab.

One text file contains one review.

I used negative (Imdb_reviews/test/neg/*.txt) and searched for bad words in reviews.

Uses: Python 3.9; Pandas; Tabulate
